# Changelog

{{version-tag-date}}

## Changes since v1.3.26

All significant changes in this version are documented here.


### ⚠ CHANGES
- Refactored UI
- Dependency updates
- Refactored underlying code
- The most perfect configuration template rules
- Refactored state management logic for improved performance and maintainability
- Fixed the issue where DNS and traffic of other devices in the local area network could not be handled in bypass mode
- TUN mode on Windows and macOS is now run as a system service, requiring only one authorization, and no longer repeatedly popping up password or UAC prompts


## 自 v1.3.26 以来的变更

本版本的所有重要变更将记录于此。


### ⚠ 变更

- 重构UI
- 依赖项更新

- 最完美的配置模板规则
- 重构状态管理逻辑以提高性能和可维护性
- 修复旁路由模式下无法处理局域网其它设备的 DNS 与流量的问题
- Windows 和 macOS 的 TUN 模式改为系统服务方式运行，只需一次授权即可，不再反复弹出密码或 UAC 提示
